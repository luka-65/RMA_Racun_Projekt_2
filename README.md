# Razvoj mobilnih aplikacija projekt 2 – Upravljanje računima
 
Ovo je android aplikacija koja omogućuje spremanje i pregled računa uz korištenje kamere mobilnog uređaja.


## Funkcionalnosti
Dodavanje računa (naziv trgovine, napomena, kategorija, datum i slika)
Fotografiranje računa pomoću kamere
Pregled svih spremljenih računa
Prikaz detalja računa
Brisanje računa uz potvrdu


## Arhitektura
Aplikacija je izrađena prema clean arhitekturi i principima uz MVVM obrazac.

Struktura slojeva:
Data – baza podataka (Room), DAO, repository
Domain – modeli i use case klase
Presentation – ViewModel i UI
